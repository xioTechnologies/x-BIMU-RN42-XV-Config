x-BIMU-RN42-XV-Config
=====================

Windows application for automated configuration of Microchip's [RN42-XV Bluetooth modules](http://www.microchip.com/wwwproducts/Devices.aspx?dDocName=en560176) for use with [x-BIMU](http://www.x-io.co.uk/products/x-bimu/).  Compiled binary available on the [x-IMU Bluetooth module](http://www.x-io.co.uk/x-bimu-bluetooth-module) web page.
